# 💎 TestifyAI - Neural Quality Engine

TestifyAI is a powerful, AI-driven automated QA auditing system that transforms the web testing experience into a high-tech audit intelligence dashboard. Built with a focus on premium aesthetics and real-time defect detection.

![TestifyAI Dashboard](https://raw.githubusercontent.com/omchavan221005/TestifyAI/main/dashboard_preview.png)

## 🚀 Key Features

*   **Audit Intelligence Dashboard**: A centralized "Command Center" featuring health score gauges and live severity tracking.
*   **Neural Defect Logging**: Real-time identification of UI inconsistencies, broken links, and form validation anomalies.
*   **Interactive History Archive**: Chronological repository of all past scan cycles with full report persistence.
*   **High-Fidelity Visuals**: A cyberpunk-inspired UI with glassmorphism, neon accents, and smooth transitions.
*   **AI-Enhanced Scans**: Leveraging advanced AI models to categorize issues and provide fix recommendations.

## 🛠️ Technology Stack

*   **Frontend**: React, Tailwind CSS, Lucide React, TanStack Query.
*   **Backend**: Node.js, Express, Puppeteer (Headless Browser), Supabase (PostgreSQL).
*   **Styling**: Cyberpunk midnight palette with custom neon utitilies.

## 🏁 Getting Started

### Prerequisites

*   Node.js v18+
*   PostgreSQL / Supabase instance

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/omchavan221005/TestifyAI.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure Environment:
   Create a `.env` file with your credentials:
   ```env
   SUPABASE_URL=your_url
   SUPABASE_KEY=your_key
   OPENAI_API_KEY=your_key
   ```

4. Launch Engine:
   ```bash
   npm run dev
   ```
